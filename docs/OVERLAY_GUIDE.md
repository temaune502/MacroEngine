# Документація TML Overlay

Оверлей (HUD) — це прозоре вікно поверх усіх інших вікон, яке дозволяє виводити текстову інформацію з макросів у реальному часі.

## Основні можливості

- Виведення декількох текстових полів (labels).
- Налаштування позиції оверлея на екрані.
- Динамічна зміна розміру шрифту для кожного поля.
- Автоматичне підлаштування розміру під вміст.
- Прозорість для миші (не заважає грі/роботі).

## Команди керування (об'єкт `ui`)

Всі команди викликаються через вбудований об'єкт `ui`.

### `ui.show()`
Показує оверлей на екрані.

### `ui.hide()`
Приховує оверлей.

### `ui.move(x, y)`
Переміщує оверлей у вказані координати екрана.
- `x`: координата по горизонталі (пікселі).
- `y`: координата по вертикалі (пікселі).

### `ui.set_text(id, text)`
Встановлює або оновлює текст для певного поля.
- `id`: унікальний ідентифікатор поля (рядок).
- `text`: текст, який потрібно вивести. Якщо передати порожній рядок `""`, поле буде приховано.

### `ui.set_font_size(id, size)`
Змінює розмір шрифту для конкретного поля.
- `id`: ідентифікатор поля.
- `size`: розмір шрифту (число). За замовчуванням — 14.

### `ui.set_size(w, h)`
Встановлює фіксований розмір вікна оверлея (зазвичай не потрібно, бо він підлаштовується сам).

## Приклад використання

```tml
let count = 0

func on_init():
    # Налаштовуємо оверлей
    ui.move(50, 50)           # Позиція в лівому верхньому куті
    ui.set_font_size("title", 20) # Робимо заголовок великим
    ui.set_font_size("stats", 12) # Статистику робимо меншою
    
    ui.show()
    ui.set_text("title", "TML MACRO ACTIVE")
    ui.set_text("stats", "Initializing...")

func on_tick(delta):
    set count = count + 1
    ui.set_text("stats", "Ticks: " + str(count))

func on_exit():
    ui.hide()
```

## Поради
1. Використовуйте `on_init()` для початкового налаштування (позиція, розміри шрифтів).
2. Якщо ви хочете "очистити" оверлей, просто передайте порожній текст у `ui.set_text("id", "")`.
3. Оверлей підтримує кодування UTF-8, тому ви можете вільно писати українською мовою.
